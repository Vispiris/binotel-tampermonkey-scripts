# Мапа фільтрів інсталяції

Це стартова мапа розділів для DXD. Вона створена на основі файлу `Список фільтрів.txt`.

## Відділ: Інсталяції

Перші розділи:

- [Загальне Інсталяції](#загальне-інсталяціі)
- Фильтр Подкл номеров
- Фильтр ТЗ
- Фильтр Удал настр
- Фильтр АЗ

## Як наповнювати

1. Додаємо статтю в `data/articles.index.json`.
2. Якщо вже є текст — створюємо Markdown-файл у `data/articles/`.
3. У полі `file` прописуємо шлях до Markdown-файлу.
4. Для пошуку оновлюємо `summary` і `tags`.

## Статуси

- `active` —[[anchor:стаття-готова]] стаття готова.
- `draft` — стаття є в списку, але текст ще треба перенести.
- `review` — статтю треба перевірити або актуалізувати.
